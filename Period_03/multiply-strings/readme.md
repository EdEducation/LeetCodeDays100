# Идея


Решение задачи "Multiply Strings" (LeetCode 43)

Оптимальный алгоритм: Умножение в столбик с использованием массива

Временная сложность: O(m * n), где m и n - длины строк
Пространственная сложность: O(m + n)

Именованный алгоритм: "Grade School Multiplication" (умножение в столбик)

Математическое обоснование:
При умножении двух чисел разрядность результата не превышает сумму разрядностей
Произведение разрядов i (из num1) и j (из num2) попадает в позиции (i+j) и (i+j+1) результата

К сожалению, алгоритма за O(1) не существует, так как необходимо обработать все разряды чисел
длиной до 200 символов, что требует как минимум O(n*m) операций в общем случае.

Однако можно использовать БПФ (Быстрое преобразование Фурье) для ускорения до O(n log n),
но это сложнее в реализации и избыточно для ограничений задачи.

## Algo

реализовать умножение в столбик с использованием массива

   1. Обработка случая с нулем

   2. Создание массива для хранения промежуточных результатов

   3. Перебор разрядов в обратном порядке

   4. Накопление произведений с учетом позиций

   5. Обработка переносов

   6. Преобразование в строку без ведущих нулей
