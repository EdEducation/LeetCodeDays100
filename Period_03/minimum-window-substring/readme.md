## TODO: реализовать алгоритм минимального окна

    1. Подсчитать частоты символов в t

    2. Использовать скользящее окно [left, right]

    3. Расширять окно, пока не найдем все символы из t

    4. Сжимать окно слева, сохраняя минимальную длину

    5. Вернуть подстроку минимальной длины или ""


**Объяснение алгоритма:**

1. **Подсчет необходимых символов** : Создаем словарь `need` с частотами символов из строки `t`.
2. **Скользящее окно** : Используем два указателя `left` и `right` для обозначения текущего окна.
3. **Расширение окна** : Двигаем `right` вправо, добавляя символы в окно и обновляя счетчики.
4. **Проверка валидности** : Когда в окне собраны все необходимые символы (с правильными частотами), пытаемся сжать окно слева, чтобы найти минимальную длину.
5. **Сжатие окна** : Двигаем `left` вправо, удаляя символы из окна, пока окно остается валидным.
6. **Отслеживание минимума** : Запоминаем минимальную найденную подстроку.
