# 2078. Two Furthest Houses With Different Colors

**Easy**

## Описание

На улице ровно расположены `n` домов, каждый из которых красиво покрашен. Вам дан **0-индексированный** целочисленный массив `colors` длиной `n`, где `colors[i]` представляет цвет `i`-го дома.

Верните **максимальное расстояние** между **двумя** домами с **разными** цветами.

Расстояние между `i`-м и `j`-м домами равно `abs(i - j)`, где `abs(x)` — это **абсолютное значение** `x`.

## Примеры

### Пример 1

![Пример 1](https://assets.leetcode.com/uploads/2021/10/31/eg1.png)

**Вход:** `colors = [1,1,1,6,1,1,1]`  
**Выход:** `3`  
**Объяснение:** Дома с разными цветами, находящиеся дальше всего — дом 0 и дом 3. Расстояние между ними: `abs(0 - 3) = 3`.

### Пример 2

![Пример 2](https://assets.leetcode.com/uploads/2021/10/31/eg2.png)

**Вход:** `colors = [1,8,3,8,3]`  
**Выход:** `4`  
**Объяснение:** Дома с разными цветами, находящиеся дальше всего — дом 0 и дом 4. Расстояние: `abs(0 - 4) = 4`.

### Пример 3

**Вход:** `colors = [0,1]`  
**Выход:** `1`  
**Объяснение:** Единственная пара домов с разными цветами — 0 и 1. Расстояние: `abs(0 - 1) = 1`.

## Ограничения

- `n == colors.length`
- `2 <= n <= 100`
- `0 <= colors[i] <= 100`
- Тестовые данные сгенерированы так, что **как минимум два** дома имеют разные цвета.

## Подсказки

1. Ограничения небольшие. Можете ли вы попробовать перебрать все пары домов?
2. Жадно, максимальное расстояние будет достигаться либо парой самого левого дома и некоторого правого дома с другим цветом, либо парой самого правого дома и некоторого левого дома с другим цветом.
